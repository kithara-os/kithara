# Kithara
Kithara is a microkernel designed to run on x86_64 architecture.

What is done:
* IPI/SIPI sequence
* Physical memory management (Buddy allocator)
* Spinlocks
* IDT/GDT/TSS

What is yet to be done:
* Scheduler
* Virtual memory management
* Processes
* IPC
* Loading init process from ramdisk
